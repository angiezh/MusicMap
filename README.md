# MusicMap

## Installation

1. Clone the repository:<br>
   git clone https://github.com/angiezh/MusicMap.git

3. Install backend dependencies:<br>
   cd backend<br>
   npm install

5. Install frontend dependencies:<br>
   cd frontend
   <br>npm install


## Running Locally

- To start the backend server:<br>
  cd backend<br>
  npm start

- To run the frontend:<br>
  cd frontend<br>
  npm start
